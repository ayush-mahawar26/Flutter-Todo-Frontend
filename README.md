
# Todo-Frontend

Todo Application , whose frontend is made up using flutter sdk with the custom backend using nodejs, express and MongoDB.



## Installation

Fork this repo , in your local 

```bash
cd - project
flutter pub get
```


    
## Tech Stack

**Client:** Flutter , Dart

**Server:** Node, Express

**Database:** MongoDB

**Hosting Platform:** render.com
